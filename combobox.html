Multi-stroke Auto-select Combo

Combo box or dropdown list behaves in such a way that it will select the item that matches its first letter as you type inside it. In a very long list, it is more convenient to type continuously to select the item that matches the word or phrase you typed (just like in Index in Help files but without that extra field!). I've whipped up this code to do just that. Try the demo below:

This combo box allows for non-existent entries to be entered into the empty first option. Try P-I-N-O-T(space)G-R-I-S or S-Y-R-A-H

Note:
- Press ESC key to reset keystrokes
- Press BACKSPACE key to delete last keystroke
- Press TAB or ENTER key to select item and set focus to next element
- Look at status bar to see what you are typing - can be disabled if you so wish.
- Works with IE/Netscape6/Mozilla
- Degrades well with browsers not supporting onkeydown (e.g. Netscape4) on select element
